Albums

The Album menu provides access to all album-specific functions in flPhoto:

  • New
  • Open...
  • Save
  • Save As...
  • Close
  • Import
  • Export
  • Print...
  • Slideshow
  • Quit

In addition, you can drag images in the album to re-order the images for printing or a slideshow.

New

Choose New to create a new photo album.

Open...

Choose Open... to open an existing photo album.

Save

Choose Save to save the current photo album.

Save As...

Choose Save As... to save the current photo album with a new name.

Close

Choose Close to close the current photo album.

Quit

Choose Quit to close all photo albums and exit the flPhoto program completely.
